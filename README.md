# Crowd CLI
CLI application aimed to provide the missing functionalities when it comes to managing Atlassian Crowd's.

## Installation
```commandline
git clone <repo>
cd crowd-cli
python3 -m venv venv
venv/bin/pip3 install -r requirements.txt
venv/bin/python3 crowd.py --help
```

## Usage
```commandline
--help
```
Works after every command.
